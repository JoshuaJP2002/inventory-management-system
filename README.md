# Inventory Management System in C#

## Overview

This project is an Inventory Management System created in C# for the CSE 310 Applied Programming course. The purpose of this software is to practice object-oriented programming concepts and file handling using C#.

The program allows users to add, view, update, and remove products from an inventory. The inventory can also be saved to and loaded from a file.

## Features

- Add products
- View inventory
- Update product quantities
- Remove products
- Save inventory to a file
- Load inventory from a file
